Milestone-2 Functional Architecture

POST /documents/upload
 → read file bytes
 → generate SHA-256
 → check hash in DB
 → reject duplicate OR save file
 → store metadata

Document Verification Flow

GET /documents/{id}/verify
 → read stored file
 → recompute hash
 → compare with DB hash
 → valid / tampered

Document Download Flow

GET /documents/{id}/download
 → authenticate user
 → role check
 → org check
 → read file from storage
 → recompute SHA-256 hash
 → compare with stored hash_key
    ├─ mismatch → BLOCK download (tampered)
    └─ match → stream file
